## Configuration section

Schema for the configuration section of the workflow

### Type: `object`

| Property | Type | Required | Possible values | Description |
| -------- | ---- | -------- | --------------- | ----------- |
| ADDITIONAL_JOBS | `object` |  | object |  |
| CONTAINER_DIR | `string` |  | string | Path to the directory where the containers are stored. |
| PROJECT_SCRATCH | `string` |  | string | Path to the directory where the project's scratch is stored. |
| LIBDIR | `string` |  | string | Path to the directory where the lib directory is, inside the repo. |
| SCRIPTDIR | `string` |  | string | Path to the directory where the runscripts are stored. |
